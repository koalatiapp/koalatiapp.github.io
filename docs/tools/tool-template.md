# Tool template

Whether it is your first time looking into Koalati tools or you just want a simple package template to build your next tool from, we recommend you look into our Tool Template, which is available on GitHub as a starting point for your tools.

The template contains all of the basics you need to get started. Just start developing in its `tool.js`, update the information in `package.json` to match your tool, and you're good to go!

[Get the Tool template on GitHub](https://github.com/koalatiapp/tool-template){: .btn .btn-purple target="_blank" }

---

## Before publishing...
Before you publish a tool built with our tool template, make sure you have updated the following files to ensure the information they contain is adequate:

- `tool.js`: implement your tool in this file (and others if need be)
- `package.json`: update it to matcch your tool's purpose and package information.
- `LICENSE`: update the license so it's under your own name (or your company's name)
