# What are Koalati tools?
Koalati tools are Node packages that are developed in a specific format in order to be compatible with Koalati's website testing platform.

The goal of a tool is to provide recommandations to web developers/designers to help them improve their websites. To do so, a tool analyzes a specific aspect of a website's pages, looking for flaws, issues and/or possible improvements.

Good tools find issues and informs the developer of those issues.
**Great tools** automatically recommend ways to fix those issues, explain why a change should be done and why it should be done that way, providing external resources for the issue at hand.

## Examples of existing tools
Here are a few examples of existing tools built by Koalati.

- **Spellcheck tool:** the Spellcheck tool extracts all relevant text content from a webpage, and runs it through a powerful spellchecking service in order to detect grammar, style and spelling mistakes. [Learn more about Koalati's spellcheck tool](https://github.com/koalatiapp/tool-spellcheck){: target="_blank" }
- **On-page SEO tool:** the SEO tool analyzes a page's meta tags, contents and URL (among other things) to help optimize and improve its reach on search engines like Google. [Learn more about Koalati's on-page SEO tool](https://github.com/koalatiapp/tool-seo){: target="_blank" }
- **Accessibility tool:** the accessibility tool uses Google Lighthouse's `accessibility` audit to detect issues and possible improvements to make websites more accessible for everyone. [Learn more about Koalati's accessibility tool](https://github.com/koalatiapp/tool-accessibility){: target="_blank" }
- **Social tool:** the social tool analyzes a page's meta tags to help optimize and improve its social sharing experience (such as sharing on Twitter or Facebook). [Learn more about Koalati's social tool](https://github.com/koalatiapp/tool-social){: target="_blank" }

To see all of Koalati's tools and librairies, check out [our GitHub](https://github.com/koalatiapp){: target="_blank" }.
