Foire aux questions

Que se passera-t-il lorsque ma période d'essai sera terminée ?

Votre période d'essai se terminera 14 jours après la création de votre compte si vous n'avez pas passé à un forfait payant.

Une fois cette période de 14 jours terminée, vos projets seront désactivés jusqu'à ce que vous passiez à un forfait payant.

Si vous aviez créé une équipe, vous ne pourrez plus n'y accéder, ni à ses projets, jusqu'à ce que vous passiez à notre plan Petite équipe ou Entreprise, ou jusqu'à ce que vous transfériez l'équipe à un utilisateur qui est sur l'un de ces plans. Pour transférer une équipe à un autre utilisateur, veuillez nous contacter.

‍

Qu'est-ce qu'un "Test de page" ?

Un test de page est défini comme une seule URL testée une fois par la fonction de test automatisé de Koalati.

Par exemple, tester une fois un site Web complet de 50 pages se traduira par des tests de 50 pages.
Tester à nouveau l'intégralité de ce site Web entraînerait 50 tests de page supplémentaires.‍

‍

Tests de page dépassant le quota

Chaque plan d'abonnement comprend un certain nombre de tests de page. Une fois ce quota de test de page atteint, chaque test de page additionnel entraînera un coût supplémentaire (en plus du coût de votre abonnement).

Ce coût basé sur l'utilisation sera facturé sur le mode de paiement actif de votre compte à la fin de chaque cycle d'utilisation. Si l'abonnement de votre compte est annulé, vous serez toujours facturé pour toute utilisation supplémentaire qui n'a pas encore été payée à la fin de votre cycle d'utilisation régulier.

Le prix par test de page supplémentaire et le quota de test de page varient en fonction de votre plan d'abonnement et peuvent être consultés sur la page Plans d'abonnement.

‍

Plusieurs tests pour la même URL dans un court laps de temps

Koalati traite généralement les demandes de test en lots pour réduire le temps de chargement et de traitement.

Si deux tests ou plus sont demandés pour la même URL dans un laps de temps de 30 secondes, ces tests compteront comme un seul test de page pour votre quota d'utilisation. La fenêtre de 30 secondes est calculée à partir de l'heure de la première requête sur la page.

Ceci est fait pour refléter le fait que nous pouvons souvent optimiser les requêtes par lots comme celle-ci dans le backend. Et si nous pouvons économiser de l'argent en regroupant les demandes, il est juste de vous permettre, à vous les utilisateurs, de bénéficier également de ces économies.

Combien de projets puis-je avoir ?

Autant que vous le souhaitez !

Il n'y a pas de limite au nombre de projets que vous pouvez avoir; seuls les projets actifs sont pris en compte dans votre quota mensuel.

Cela vous permet de conserver une archive du statut et de l'avancement du processus de contrôle qualité pour chaque projet, et de reprendre là où vous vous étiez rendus quand vous le souhaitez.

‍

Puis-je rejoindre une équipe en tant qu'utilisateur gratuit ?

Oui! Tous les utilisateurs peut rejoindre des équipes, peu importe leur plan d'abonnement.

Rejoindre une équipe vous donnera accès aux fonctionnalités et quotas de cette équipe, mais uniquement pour les projets de l'équipe.

Vos projets personnels seront toujours limités aux quotas et aux fonctionnalités de votre propre plan.

Combien d'équipes puis-je créer ?

Chaque compte avec le privilège de création d'équipe peut créer autant d'équipes que désirées.

Cependant, veuillez noter que chaque équipe que vous créez partage les quotas d'utilisation du compte qui l'a créée.

Existe-t-il des plans avec des quotas plus élevés ?

Il n'y a pas de forfaits avec des quotas plus élevés.

Cependant, vous pouvez autoriser l'utilisation à dépasser le quota de votre forfait et définir une limite de dépenses afin de contrôler l'utilisation.

Cette option est désactivée par défaut afin d'éviter des frais accidentels et/ou indésirables, mais peut être activée dans les paramètres d'abonnement de l'application.

Le coût de l'utilisation hors quota varie en fonction de votre plan d'abonnement:

  • Plan Solo: 0,00118056$ / test de page supplémentaire.
  • Plan Petite Équipe: 0,00118056$ / test de page supplémentaire.
  • Plan Entreprise: 0,00079861$ / test de page supplémentaire.
